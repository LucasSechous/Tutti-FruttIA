# Tutti-FruttIA
Este es nuestro proyecto del 4to semestre de la facultad(CEI). Junta las materias Diseño y desarrollo de aplicaciones con Ingeniería de Software, para crear una app similar al conocido juego tutti frutti pero con un juez que se basa en IA para verificar y validar las palabras de los usuarios que estén jugando además de elegir la letra al azar
